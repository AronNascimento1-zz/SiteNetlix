🧑‍💻 Instalações e usos

Clone ou faça o download desse repositório:

```
# Clone o repositório
$ git clone git@github.com:AronNascimento1/SiteNetlix.git
```

Acesse a pasta server e faça as instalações:

```
# Instale as dependencias
$ npm install

# Rode o projeto
$ npm start
```
